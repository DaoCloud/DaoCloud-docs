# 消息模板

消息模板功能支持自定义消息模板的内容，并可邮件、企业微信、钉钉、Webhook 的形式通知指定的对象。

## 创建消息模板

1. 在左侧导航栏中，选择`告警中心` -> `消息模板`，点击`新建消息模板`按钮。

  ![消息模板](../../images/template01.png)

2. 填写模板内容。

  ![消息模板](../../images/template02.png)

## 编辑或删除消息模板

在模板列表中，点击最右侧的 `...`按钮，选择相应菜单项。

  ![编辑按钮](../../images/template03.png)

> 请注意，删除模板后无法恢复，请谨慎操作。
