# 选择工作空间

首次进入微服务治理模块时，首先必须选择一个[工作空间](../../ghippo/user-guide/workspace/workspace.md)。

![选择工作空间](https://docs.daocloud.io/daocloud-docs-images/docs/skoala/images/workspace.png)

如需更改当前所在的工作空间，可以在左侧边栏点击更换图标重新选择工作空间。

![更改工作空间](https://docs.daocloud.io/daocloud-docs-images/docs/skoala/images/change-ws.png)

!!! note

    如果当前没有可选的工作空间，需要先联系超级管理员创建一个工作空间。
