# 创建模型

*[Hydra]: 大模型服务平台的开发代号

模型管理用于在当前工作空间内维护自定义模型，并为后续部署准备部署模板与模型权重文件。
如需使用平台预置的公共模型，请前往[模型广场](../index.md)查看与部署。

## 查看模型列表

1. 进入大模型服务平台，在左侧导航栏展开 **模型服务**，点击 **模型管理**。

2. 在模型管理列表中，可查看当前工作空间内的自定义模型，并按 **模型名称** 搜索，或按 **创建时间** 排序。

    | 列名 | 说明 |
    | --- | --- |
    | 模型名称 | 点击名称进入模型详情 |
    | 模型标签 | 模型能力类型，如文本生成、图片生成等 |
    | 模型描述 | 按当前界面语言展示中文或英文说明 |
    | 创建时间 | 模型创建时间 |

3. 在列表右侧 **┇** 菜单中，可执行 **部署**、**更新**、**删除** 操作。

## 创建模型

1. 在 **模型管理** 页面中，点击 **创建模型**。

2. 在创建页面填写参数信息后，点击 **确定**，即创建模型成功，返回到模型管理列表。

    | 参数项 | 约束 / 说明 | 备注 |
    | --- | --- | --- |
    | 模型名称 | 必填<br>**长度**：最长 63 个字符<br>**字符**：仅支持小写字母、数字、短横线（-）或点（.）<br>**规则**：必须以字母或数字开头和结尾 | |
    | 模型标签 | 必填，可多选 | 用于标识模型能力类型 |
    | 模型图标 | 可选 | 支持本地上传，或输入以 `https://` 开头的图片 URL；支持 SVG、PNG、JPG、JPEG 格式，文件小于 40 KB，建议尺寸 100 x 100 像素 |
    | 模型说明 | 可选 | 中文、英文各最多 800 字 |

## 编辑模型

1. 在模型列表中，点击目标模型右侧的 **┇** 菜单，选择 **更新**。

2. 在编辑页面修改模型名称、模型标签、模型图标或模型说明后，点击 **确定** 保存变更。

    也可在模型详情页右上角 **┇** 菜单中选择 **更新**，进入同一编辑页面。

## 查看模型详情

1. 在模型列表中，点击目标 **模型名称**，进入模型详情页面。

2. 详情页上方展示模型名称、模型标签、模型描述、创建时间等基本信息。

3. 详情页下方通过 **部署模板**、**模型权重文件** 两个 tab 管理部署相关配置。

    详情页右上角支持 **部署**，以及 **更新**、**删除** 操作。

## 管理部署模板

部署模板用于预置资源配置与运行配置。部署自定义模型时，可直接选用模板，减少重复填写。

### 查看部署模板

1. 在模型详情页面，选择 **部署模板** tab。

2. 可按 **模板名称**、**运行时框架** 筛选模板，并按 **创建时间** 排序。

    | 列名 | 说明 |
    | --- | --- |
    | 模板名称 | 部署时选用的模板标识 |
    | 资源规格 | CPU、内存配置 |
    | GPU | GPU 类型与卡数 |
    | 运行时 | 推理运行时框架 |
    | 描述 | 模板说明 |
    | 创建时间 | 模板创建时间 |

### 创建部署模板

1. 在 **部署模板** tab 中，点击 **创建**。

2. 依次填写 **基本信息**、**资源配置**、**运行配置** 后，点击 **确定**，即创建部署模板成功，返回到模型详情页面。

    | 参数项 | 约束 / 说明 | 备注 |
    | --- | --- | --- |
    | 模板名称 | 必填<br>**长度**：最多 63 个字符<br>**字符**：仅支持小写字母、数字、短横线（-）或点（.）<br>**规则**：必须以字母或数字开头和结尾 | |
    | 描述 | 可选 | 最多 800 字 |
    | CPU / 内存 | 资源配置 | |
    | GPU | 可选 | 含 GPU 类型、物理卡数量、算力与显存 |
    | 运行时框架 | 选择推理运行时 | 切换框架时会自动带出默认启动命令 |
    | 启动命令 | 可按需修改 | |
    | 环境变量 | 可选 | 以键值对形式配置 |

    如需使用自定义推理运行时，请先在运维侧完成运行时配置，详见[自定义大模型推理运行时](../user-guides/custom-runtime.md)。

### 更新部署模板

1. 在 **部署模板** tab 中，点击目标模板右侧的 **┇** 菜单，选择 **更新**。

2. 在编辑页面修改参数后，点击 **确定** 保存变更。

### 删除部署模板

1. 在 **部署模板** tab 中，点击目标模板右侧的 **┇** 菜单，选择 **删除**。

2. 在弹窗中确认删除操作。

    !!! note

        删除后不可恢复，请谨慎操作。

## 管理模型权重文件

模型权重文件用于将集群上的权重目录与容器内挂载路径关联到当前模型。部署自定义模型时，可据此挂载对应权重。

!!! note

    目标集群需已安装 Hydra Agent，且已创建[文件存储](../file-storage/storage.md)。
    权重文件需已上传到文件存储对应目录，或通过[远端文件预热](../file-storage/file-preheat.md)同步到集群。
    如需了解文件上传方式，请参考[管理文件](../file-storage/file-manage.md)。

### 查看模型权重文件

1. 在模型详情页面，选择 **模型权重文件** tab。

2. 可按 **集群** 筛选权重记录，并按集群名称排序。

    | 列名 | 说明 |
    | --- | --- |
    | 集群 | 权重所在集群 |
    | 版本标识 | 同一模型在同一集群下的权重版本标识 |
    | 存储类型 | 当前支持 **文件存储** |
    | 存储目录 | 文件存储中的目录路径，可跳转至文件存储详情 |
    | 挂载路径 | 权重在推理容器内的挂载路径 |
    | 更新时间 | 最近一次更新时间 |

### 创建模型权重文件

1. 在 **模型权重文件** tab 中，点击 **创建**。

2. 填写参数信息后，点击 **确定**，即创建模型权重文件成功，返回到模型详情页面。

    | 参数项 | 约束 / 说明 | 备注 |
    | --- | --- | --- |
    | 集群 | 必填 | 仅可选择已安装 Agent 且已创建文件存储的集群；若当前集群尚未创建文件存储，可点击 **去创建** 跳转至文件存储创建页面 |
    | 版本标识 | 必填 | 用于区分同一集群下的不同权重版本 |
    | 存储类型 | 必填 | 当前仅支持 **文件存储** |
    | 存储目录 | 必填 | 可手动输入，或点击 **选择目录** 从文件存储中选取；也可通过 **去创建** 跳转管理文件 |
    | 挂载路径 | 必填 | 权重在容器内的挂载路径，默认示例为 `/mnt/path` |

### 更新模型权重文件

1. 在 **模型权重文件** tab 中，点击目标权重记录右侧的 **┇** 菜单，选择 **更新**。

2. 在编辑页面修改参数后，点击 **确定** 保存变更。

    !!! note

        已被模型服务挂载的权重不支持编辑。

### 删除模型权重文件

1. 在 **模型权重文件** tab 中，点击目标权重记录右侧的 **┇** 菜单，选择 **删除**。

2. 在弹窗中确认删除操作。

    !!! note

        已被模型服务挂载的权重不支持删除。

## 部署自定义模型

1. 在模型列表或模型详情页中，点击 **部署**。

2. 进入模型部署页面后，将 **模型来源** 选择为 **自定义模型**，并选用已配置的部署模板与模型权重文件。

    部署表单参数说明请参考[部署新模型](../deploy/deploy.md)。
    v0.13.1 之前版本的操作说明，也以该文档为准；自定义来源、部署模板、模型权重、分布式推理、队列调度等能力，将在后续文档中补充。

## 删除模型

1. 在模型列表或模型详情页中，点击 **┇** 菜单，选择 **删除**。

2. 系统会校验当前模型是否存在关联的模型服务：

    - 若存在关联服务，请先删除对应模型服务，再删除模型；
    - 校验通过后，输入模型名称并确认删除。

    !!! note

        删除后不可恢复，请谨慎操作。
