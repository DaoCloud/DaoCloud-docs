# 大模型服务平台 Release Notes

本页列出大模型服务平台的 Release Notes，便于您了解各版本的演进路径和特性变化。

*[hydra]: DaoCloud 大模型服务平台的内部开发代号

## 2026-05-31

### v0.15.0

- **新增** 大模型微调功能，支持数据集管理、微调任务管理、模型评估和导出
- **新增** 模型部署支持容错检查，在模型正式部署前自动检查关键硬件及软件环境，提前发现问题
- **新增** 模型部署/MaaS 模型支持 token 权重配置，以保证 token 配额的公平分配
- **新增** API Key 支持启用/禁用，用户可以根据需要选择是否启用 API Key
- **新增** 大语言模型支持缓存命中 tokens 计量
- **新增** 运维管理支持 workspace token 配额设置，用户可以根据需要为每个 workspace 设置 token 限额
- **新增** 运维管理支持网关安全策略管理和审计日志，用户可以根据需要配置网关的安全策略，如 敏感词检测、模型黑白名单等
- **新增** MaaS 模型支持语音转文字、OCR、rerank、embedding 等多模态模型转发
- **优化** 模型广场管理支持删除模型

!!! note

    hydra 从 0.15.0 版本开始，在 knoway 网关中集成了 higress，以提供 AI 安全、token 限额等能力，详情参见[升级注意事项](upgrade-notes.md)。

## 2026-04-30

### v0.14.0

- **新增** API Key 支持有效期设置
- **新增** API Key 支持 token 配额设置
- **优化** API Key 管理权限：每个用户创建和管理自己的 API Key
- **优化** 模型体验仅支持选择自己创建的 API Key 进行鉴权
