# 什么是 Kafka

Kafka 模块是一款基于开源软件 Kafka 提供的分布式消息队列服务。
DaoCloud 为其开发了简单易用的图形化界面，向用户提供计算、存储和带宽资源独占的 Kafka 专享实例。

Kafka 是一个拥有高吞吐、可持久化、可水平扩展，支持流式数据处理等多种特性的分布式消息流处理中间件，采用分布式消息发布与订阅机制，在日志收集、流式数据传输、在线/离线系统分析、实时监控等领域有广泛的应用。

目前支持的特性如下：  

- 创建/更新/删除 Kafka 实例
- 可靠性保证机制  
- 支持自定义参数配置  
- 支持集群高可用  
- 多语言客户端支持  
- 简单易用的图形界面
