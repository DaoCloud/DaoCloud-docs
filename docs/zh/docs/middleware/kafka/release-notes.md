# Kafka 消息队列 Release Notes

本页列出 Kafka 消息队列的 Release Notes，便于您了解各版本的演进路径和特性变化

*[mcamel-kafka]: mcamel 是 DaoCloud 所有中间件的开发代号，kafka 是提供分布式消息队列服务的中间件
