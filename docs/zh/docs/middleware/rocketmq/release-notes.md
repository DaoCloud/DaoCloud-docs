# RocketMQ 消息队列 Release Notes

本页列出 RocketMQ 消息队列的 Release Notes，便于您了解各版本的演进路径和特性变化。

- **修复** 容器组列表未展示实例中所有相关的容器组

### v0.10.0

### v0.5.0
