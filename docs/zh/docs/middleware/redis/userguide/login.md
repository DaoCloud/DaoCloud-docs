# 首次使用 Redis

请提前联系管理员在`容器管理` -> `Helm 应用`[安装好 redis-operator](./install.md) 后，执行以下操作。

1. 在左侧导航栏中选择`中间件` -> `Redis 缓存服务`。

    ![](../images/redis01.png)

2. 在弹窗中选择一个工作空间后，点击`确认`。

    ![](../images/redis02.png)

    !!! note
    
        如果未出现弹窗，可手动点击左上角图标选择一个工作空间。

3. 首次登录时，可以点击`立即部署`来创建实例。此后登录时，将显示所有 Redis 实例及其运行状态等信息。

    ![](../images/redis03.png)
