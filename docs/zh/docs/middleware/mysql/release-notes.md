# MySQL Release Notes

本页列出 MySQL 数据库的 Release Notes，便于您了解各版本的演进路径和特性变化。

*[mcamel-mysql]: mcamel 是 DaoCloud 所有中间件的开发代号，mysql 是应用最广泛的关系型数据库

## 2024-09-30

### v0.22.0

- **新增** 支持手动切换主从节点
- **新增** MGR 实例支持配置 Router 节点
- **修复** 部分操作无审计日志的问题

### v0.21.0
