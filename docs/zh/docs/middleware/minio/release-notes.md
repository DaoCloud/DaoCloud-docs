# MinIO 对象存储 Release Notes

本页列出 MinIO 对象存储的 Release Notes，便于您了解各版本的演进路径和特性变化。

*[mcamel-minio]: mcamel 是 DaoCloud 所有中间件的开发代号，minio 是一款热门、轻量的对象存储中间件
