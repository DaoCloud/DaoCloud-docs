# 首次进入 MinIO

请提前联系管理员在`容器管理` -> `Helm 应用`[安装好 MinIO-operator](../quickstart/install.md) 后，执行以下操作。

1. 在左侧导航栏中选择`中间件` -> `MinIO 存储`。

    ![](../images/view01.png)

2. 在弹窗中选择一个工作空间后，点击`确认`。

    ![](../images/login02.png)

    !!! note
    
        如果未出现弹窗，可手动点击左上角图标选择一个工作空间。

3. 初次使用时，可以点击`立即部署`来创建 MinIO 实例。

    ![](../images/what03.png)
