# Unifabric Release Notes

## 2025-11-05

### v0.1.0

- **新增** 分层拓扑展示，支持展示 Spine、Leaf 交换机、超节点组和存储设备的连接架构。
- **新增** 连接状态指示能力，支持实时显示设备之间的连接状态，连接中断时链路自动消失。
- **新增** Spine/Leaf 交换机监控，支持展示交换机运行数据与监控指标。
- **新增** 计算节点监控，支持显示计算节点状态与性能信息。
- **新增** 存储设备展示能力，支持展示存储设备的连接状态。
- **新增** 算力交换机列表，支持展示交换机角色、管理 IP、端口状态、TX 带宽、利用率与启动时间等信息。
- **新增** 存储交换机列表，支持展示管理 IP、端口状态、TX 带宽与利用率等信息。
- **新增** 节点列表，支持展示节点基本信息、RDMA 数量、Pod 数量与带宽利用率。
- **新增** 拓扑跳转能力，支持点击超节点区域查看对应超节点组内部拓扑视图。
- **新增** 集群切换能力，支持多集群切换展示。
- **新增** 超节点切换能力，支持超节点组视角切换。
- **新增** 超节点分组视图，支持展示指定超节点组内部互联结构。
- **新增** 节点状态展示，支持展示节点健康状态。
- **新增** GPU 互联细节展示，支持展示不同超节点之间的 GPU 链路连接。
- **新增** 链路性能分析，支持点击任意链路路径查看实时链路性能指标。
- **新增** 链路列表视图，支持展示链路节点名、GPU 设备 ID、链路 ID、RX/TX 速率、网络错误（UE）与拥塞（CE）等信息。
