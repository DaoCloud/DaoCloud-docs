# 虚拟机 Release Notes

本页列出虚拟机的 Release Notes，便于您了解各版本的演进路径和特性变化。

## 2025-08-31

### v0.19.0

### 功能

- **新增** 虚拟机配置网络时禁止不同网卡使用相同 MultusCR 和相同的 IP 池
- **新增** 虚拟机创建/更新时支持设置 DNS
- **优化** 适配 spiderpool v1.0.2
- **修复** 修复虚拟机部分时间内容有误问题
- **修复** 修复虚拟机存储扩缩容报错问题


## 2025-07-30

### v0.18.1

- **新增** 虚拟机列表 API 支持 IP 搜索
- **优化** VirtNest 对接 Insight(`metrics`, `log`, `otel tracing`) 来满足开发最佳实践
- **优化** 用脚本和文档来支持虚拟机的定时快照+快照备份恢复
- **修复** 批量创建虚拟机报错问题

## 2025-04-30

### v0.17.0

- **升级** GPU 选择界面的交互体验

## 2025-02-28

### v0.16.0

- **新增** 界面对存储是否支持热插拔进行提示
- **修复** 虚拟机集群内冷迁移镜像地址为空的问题
- **修复** 虚拟机概览下硬盘数量不对的问题

- **修复** 带 GPU 的虚拟机创建不成功问题
