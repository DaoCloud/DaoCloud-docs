# 流水线分组

通过分组可以对流水线进行分类，方便企业可以管理流水线。

## 分组管理

每个工作空间可以创建多个流水线分组

1. 前往 __流水线__，点击 `分组管理` 按钮

    ![group1](../../../images/group1.jpg)

2. 在分组管理界面输入分组名称即可创建

    ![group2](../../../images/group2.jpg)

3. 也可以对分组进行删除，删除该分组后，该分组内的流水线的分组也会被删除

    ![group3](../../../images/group3.jpg)
