# 安装教程

DCE 5.0 分为社区版和商业版。社区版可免费使用，包括全局管理、容器管理、可观测性三大模块。商业版可按需订购微服务、网格、多云、中间件等所有模块。更多详情可参考[什么是 DCE 5.0](../dce/what-is-dce.md)及其[适用场景](../dce/scenario.md)。
