# 微服务引擎权限说明

[微服务引擎](../../skoala/intro/features.md)包括微服务治理中心和微服务网关两部分。微服务引擎支持三种用户角色：

- Workspace Admin
- Workspace Editor
- Workspace Viewer

每种角色具有不同的权限，具体说明如下。

<!--
有权限使用`&check;`，无权限使用`&cross;`
-->
