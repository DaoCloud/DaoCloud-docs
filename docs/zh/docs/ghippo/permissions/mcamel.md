# 中间件数据服务权限说明

[中间件数据服务](../../middleware/index.md)包括了许多精选的中间件：MySQL、Redis、MongoDB、PostgreSQL、Elasticsearch、Kafka、RabbitMQ、RocketMQ、MinIO。
中间件数据服务支持三种用户角色：

- Workspace Admin
- Workspace Editor
- Workspace Viewer

每种角色具有不同的权限，具体说明如下。

<!--
有权限使用 `&check;`，无权限使用 `&cross;`
-->
