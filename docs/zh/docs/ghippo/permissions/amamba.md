# 应用工作台权限说明

[应用工作台](../../amamba/01ProductBrief/WhatisAmamba.md)支持三种用户角色：

- Workspace Admin
- Workspace Editor
- Workspace Viewer

每种角色具有不同的权限，具体说明如下。
