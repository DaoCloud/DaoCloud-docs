=== "AMD64"

    执行以下命令校验安装包：

    ```sh

    ```

    校验成功会打印：


## 安装

成功校验离线包之后，

=== "AMD64"

    执行以下命令解压缩 tar 包：

    ```sh
    tar -zxvf offline-community-v0.9.0-amd64.tar
    ```

=== "ARM64"

    执行以下命令解压缩 tar 包：

    ```sh
    tar -zxvf offline-community-v0.9.0-arm64.tar
    ```
