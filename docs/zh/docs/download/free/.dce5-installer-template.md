## 校验

进入离线安装包下载目录。执行以下命令校验安装包：

```sh
echo "{{pkg.checksum}}" | sha512sum -c
```

校验成功会打印：

```none
{{pkg.fileName}}: OK
```

## 安装

成功校验离线包之后，解压缩 tar 包：

```sh
tar -zxvf {{pkg.fileName}}
```

然后参阅[社区版安装流程](../../install/community/k8s/online.md#_2)进行安装。

成功安装之后请[申请免费社区体验](../../dce/license0.md)。
