## 安装

成功校验离线包之后，

=== "AMD64"

    执行以下命令解压缩 tar 包：

    ```sh
    tar -zxvf offline-community-v0.3.30-amd64.tar
    ```

=== "ARM64"

    执行以下命令解压缩 tar 包：

