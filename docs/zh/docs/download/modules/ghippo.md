## 校验

在下载离线安装包和校验文件的目录，以 `v0.13.2_amd64` 为例，执行以下命令校验完整性：

```sh
echo "$(cat ghippo_v0.13.2_amd64_checksum.sha512sum)" | sha512sum -c
```

校验成功后打印结果类似于：

```none
ghippo_v0.13.2_amd64.tar: ok
```
