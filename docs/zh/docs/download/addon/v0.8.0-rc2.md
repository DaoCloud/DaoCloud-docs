# 下载 Addon 离线包

有关离线 Addon 的使用，可以在 clusterConfig.yaml 中进行配置，请查阅 [DCE 5.0 安装流程](../../install/intro.md#_3)。

## 使用

DCE 5.0 支持 Addon 离线化，下载离线包后，打开 clusterConfig.yaml，修改 `addonOfflinePackagePath` 字段，指定 Addon 所在的路径。
