## 校验

进入离线安装包下载目录。

=== "AMD64"

    执行以下命令校验安装包：

    ```sh
    echo "{{amd.checksum}}" | sha512sum -c
    ```

    校验成功会打印：

    ```none
    {{amd.fileName}}: OK
    ```

=== "ARM64"

    执行以下命令校验安装包：

    ```sh
    echo "{{arm.checksum}}" | sha512sum -c
    ```

    校验成功会打印：

    ```none
    {{arm.fileName}}: OK
    ```

## 安装

成功校验离线包之后，

=== "AMD64"

    执行以下命令解压缩 tar 包：

    ```sh
    tar -zxvf {{amd.fileName}}
    ```

=== "ARM64"

    执行以下命令解压缩 tar 包：

    ```sh
    tar -zxvf {{arm.fileName}}
    ```
