## 校验

进入离线安装包下载目录。执行以下命令校验安装包：

```sh
echo "{{pkg.checksum}}" | sha512sum -c
```

校验成功会打印：

```none
{{pkg.fileName}}: OK
```

## 安装

成功校验离线包之后，解压缩 tar 包：

```sh
tar -zxvf {{pkg.fileName}}
```

成功校验离线包之后，请参阅[商业版安装流程](../../install/commercial/start-install.md)进行安装。

成功安装之后请进行[正版授权](https://qingflow.com/f/e3291647)。
