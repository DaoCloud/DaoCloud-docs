=== "AMD64"

    执行以下命令校验安装包：


    校验成功会打印：

    ```none
    offline-centos7-v0.3.30-amd64.tar: OK
    ```

## 安装

成功校验离线包之后，

=== "AMD64"

    执行以下命令解压缩 tar 包：

    ```sh
    tar -zxvf offline-centos7-v0.3.30-amd64.tar
    ```

=== "ARM64"

    执行以下命令解压缩 tar 包：

    ```sh
    tar -zxvf offline-kylin-v10sp2-v0.3.30-arm64.tar
    ```
