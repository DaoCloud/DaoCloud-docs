# Kubernetes Versions Supported by Middleware

Currently, the following middleware has been officially released:

- RabbitMQ
- Elasticsearch
- MySQL
- Redis
- Kafka
- MinIO
- PostgreSQL
- MongoDB
- RocketMQ
