- Use OpenTelemetry Zero-Code to Receive SkyWalking Trace Data: [Receive SkyWalking Agent Trace Data](./sw-to-otel.md)
- Implement Nginx Ingress Trace Tracking: [Integrate Nginx Ingress Trace Tracking](./ingress-otel.md)
