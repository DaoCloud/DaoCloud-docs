# Kubernetes Related Resources

This page lists some Kubernetes-related website resources for easy retrieval and reference.

## K8s Official

- [Kubernetes Website](https://kubernetes.io/): User manual and reference documentation for end users
- [Kubernetes Contributor Website](https://www.kubernetes.dev/): Community organization structure, monthly meetings and activities information for contributors
- [Kubernetes Weekly Development Updates](https://lwkd.info/)
- [Kubernetes Community Common Labels](https://prow.k8s.io/command-help): Explains usage of /lgtm, /approve, /close, /remove- labels
- [Kubernetes News](https://kubenews.net/): Lists some K8s-related websites
- [CNCF Glossary](https://glossary.cncf.io/): Common CNCF terminology, currently supporting 14 languages

## Contribution Statistics

- [k8s.devstats Personal Contribution Stats](https://k8s.devstats.cncf.io/d/13/developer-activity-counts-by-repository-group?orgId=1&var-period_name=Last%20year&var-metric=contributions&var-repogroup_name=SIG%20Docs&var-repo_name=kubernetes%2Fkubernetes&var-country_name=All)
- [LFX CNCF All Projects Contribution Stats](https://openprofile.dev/my-insights)
- [Stackalytics Contribution Stats](https://www.stackalytics.io/cncf)
- [OSS Insight](https://ossinsight.io/): Statistics for GitHub project contributions and activity

## Learning and Conference Activities

- [killercoda Online Learning and Demo Environment](https://killercoda.com/)
- [CNCF Official Website](https://www.cncf.io/): K8s belongs to CNCF, one of its graduated projects
- [Linux Foundation Events](https://events.linuxfoundation.org/): Contains KubeCon, KCD, KubeDay and other event information
- [Speaker and Program Committee Matters](https://sessionize.com/)
- [Digital Badges Badge](https://www.credly.com/users/michael-yao0422)

## Related Projects

- [istio.io Official Website](https://istio.io/): #1 service mesh, CNCF graduated project
- [Apply to Join CNCF Landscape](https://github.com/cncf/landscape/pull/3380): Submit a PR following the template
- [Apply to Join CNCF Sandbox](https://github.com/cncf/sandbox/issues): Submit an Issue to apply for Sandbox, TOC committee reviews, review period about 1-3 months
- [CNCF Sandbox Onboarding](https://github.com/cncf/toc/issues): After TOC committee vote passes, complete onboarding procedures here
- [Popular Projects Donated to CNCF](https://github.com/cncf/toc/pull/827)
