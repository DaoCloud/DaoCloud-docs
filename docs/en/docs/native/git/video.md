# Video Recording and Editing Tutorial

This article takes QuickTime and剪映 (JianYing) on MacOS as examples to briefly explain the general steps of video recording and editing.

## Prepare Tools

Screen recording software just needs to:

1. Export video without watermark
2. Support microphone input

So you can use your commonly used screen recording software, or use the recommended software below:

- MacOS: QuickTime Player

    This is a built-in screen recording software, no download needed.
    Advantages: no time limit, no watermark on video background, supports microphone input, simple and easy to use.
    Disadvantages: features are somewhat limited, but sufficient for recording product operation videos.

    !!! tip

        QuickTime Player doesn't support system sound recording by default.
        To record system sound, you need to install the plugin [Soundflower](https://soundflower.en.softonic.com/mac).

- Windows: [Bandicam](https://www.bandicam.cn/)

    Screen recording software for Windows, powerful features.
    Free version can record up to 10 minutes and exported video will have watermark. Using [other versions](https://pan.baidu.com/s/1454QrpmxV2tyd4lM2sR_dQ) can avoid this issue.

- MacOS/Windows: [OBS Studio](https://obsproject.com/)
