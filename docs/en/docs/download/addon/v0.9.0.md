# Download Addons

This page allows you to download the addons for DCE 5.0.
To use the addons, you can configure it in the clusterConfig.yaml file. Please refer to the [DCE 5.0 Installation Process](../../install/index.md#_3) for more information.

## Usage

DCE 5.0 supports offline Addon installation. After downloading the offline package, open clusterConfig.yaml and modify the `addonOfflinePackagePath` field to specify the path of the Addon.
