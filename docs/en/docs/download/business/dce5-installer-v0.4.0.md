## Install

After successfully verifying the offline package,

=== "AMD64"

     run the following command to extract the tarball:

     ```sh
     tar -zxvf offline-centos7-v0.4.0-amd64.tar
     ```

     ```sh
     tar -zxvf offline-kylin-v10sp2-v0.4.0-arm64.tar
     ```
